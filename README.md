# Greedy Best-First Search Route Planner

## Overview
A route planning application that uses the Greedy Best-First Search (GBFS) algorithm to find paths between cities.

## Features
- Interactive city selection
- Route visualization
- Step-by-step search animation
- Heuristic-based path finding
- Exploration log
- Performance statistics

## Technologies Used
- HTML
- CSS
- JavaScript
- Python
- Greedy Best-First Search Algorithm

## How to Run

### Web Version
Open index.html in a browser.

### Python Version
python gbfs_planner.py

## Algorithm
The project uses Greedy Best-First Search with Haversine distance as the heuristic function.
